=============================================
>> DUNGEON MANAGEMENT SYSTEM

> Required Features:
    - "Modern Tools"
    - OOP Features
    - GUI
    - Must have concepts of encapsulation, inheritance, polymorphism, abstraction and interface
    - System interaction: Add, Update and delete events
    - Storage: Must read and write files from drive


 > Dungeon System:
    - Create and procedurally generate dungeons
    - Update functionalities of a dungeon
    - Simulate a Dungeon raid with a character
    - Create and Manage characters on your own
    - Create and Manage dungeon rooms on your own
    - Manage active dungeons and cleared dungeons
    - Manage active modifiers in a dungeon

> Project Overview:
    - Main Menu:
        - Simulate
        - Create Dungeons
        - Create Player
        - Manage
            - Dungeons
            - Dungeon Rooms
            - Players
            - History
    - Simulate
        - Generation: Shows real time generation of a dungeon
        - Raid: Shows real time raid of a dungeon
    - Create Dungeon
        - Has a lot of text fields for dungeon modifiers
        - Can randomize everything with a button click
    - Create Player
        - Has a lot of text fields for stats
        - Can randomize everything with a button click
